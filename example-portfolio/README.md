# example-portfolio

## How to run locally

```sh
cd path/to/example-portfolio

python3 --version # ensure you're running python 3

python3 -m venv venv # create a new virtual environment, save it to folder named 'venv'

. venv/bin/activate # activate virtual environment

pip install -r requirements.txt # install all of the required application dependencies, including flask

flask run # start the app!
```
